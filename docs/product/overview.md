# サービス概要

## サービス名

（未確定）

## 解決する課題

| ターゲット | 課題 |
|----------|------|
| IT知識はないが作りたい意欲がある人 | 作りたいが何を作ればいいかわからない |
| IT知識があり作ることは好きだが案がない人 | 事業案・プロダクトアイデアを思いつけない |
| 就活実績を作りたい学生 | ポートフォリオになるプロジェクトに参加したい |

## 解決手段

プロダクトアイデアを持つ人と、それを作りたい人をマッチングするプラットフォーム。

## なぜ作るか

- 作者自身も最近作りたいプロダクトが思いつかず、このようなサービスを必要と感じた
- AIエージェントの発達により個人開発のハードルが下がり、需要が高まっている
- SNS 上でバイブコーディングなど個人開発投稿が増えており、市場の盛り上がりがある

## リリース方針

- 実際のリリースは任意。まずガクチカとして使える成果物を作ることを優先する
- MVP の完成後にリリース可否を判断する
