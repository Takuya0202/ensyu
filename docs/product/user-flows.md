# ユーザーフロー

## 前提

MVP スコープのフローのみ記載する。未確定の部分は（未確定）と明示する。

---

## フロー 1: プロダクト案を投稿する

```
[ログイン済みユーザー]
  └─> 投稿フォームを開く
        ├─ タイトル
        ├─ 概要・説明
        └─ 必要なスキル・役割（未確定）
  └─> 投稿 → 案が一覧に表示される
```

---

## フロー 2: 案に参加申請する

```
[ログイン済みユーザー]
  └─> 投稿一覧を見る
  └─> 気になる案の詳細ページへ
  └─> 参加申請ボタンを押す
  └─> 投稿者に通知が届く（未確定）
  └─> 投稿者が承認 / 否認する（未確定）
```

---

## フロー 3: DM でやり取りする

```
[ユーザー A → ユーザー B]
  └─> B のプロフィール or 投稿詳細から DM を開始
  └─> メッセージを送受信する
```

---

## フロー 4: チームトークを使う

```
[プロジェクトメンバー全員]
  └─> 参加申請が承認されるとチャンネルに追加（未確定）
  └─> チームチャンネルでメッセージを送受信する
```

---

## 画面一覧（暫定）

| 画面 | 説明 |
|------|------|
| ホーム / 一覧 | プロダクト案の一覧 |
| 投稿詳細 | 案の詳細・参加申請ボタン |
| 投稿フォーム | 新規案の作成 |
| DM | 1対1のメッセージ |
| チームトーク | プロジェクトメンバー全員のチャット |
| プロフィール | ユーザー情報 |
