# MVP 機能一覧

## MVP に含める機能

| 機能 | 概要 |
|------|------|
| プロダクト案の投稿・募集 | アイデアを持つユーザーが案を投稿し、参加者を募る |
| 参加申請 | 案に対して「参加したい」と申請する |
| トーク（DM） | ユーザー間のダイレクトメッセージ |
| トーク（チーム） | 同じプロジェクトのメンバー間のグループチャット |

## MVP から除外する機能（後回し）

| 機能 | 除外理由 |
|------|---------|
| 実績機能 | MVP のスコープ外 |
| いいね機能 | 申請機能で代替できる |
| 募集要項の充実 | 最初はシンプルな投稿フォームで十分 |
| コミュニティ規約 | 運用しながら調整が必要 |
