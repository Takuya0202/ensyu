# MVP 機能一覧

## MVP に含める機能

| 機能 | 概要 |
|------|------|
| プロダクト案の投稿・募集 | アイデアを持つユーザーが案を投稿し、参加者を募る |
| 参加申請 | 案に対して「参加したい」と申請する（承認/却下） |

## MVP から除外する機能（後回し）

| 機能 | 除外理由 |
|------|---------|
| トーク（DM） | 工数が大きく、コアフロー完成後に追加する |
| トーク（チーム） | 工数が大きく、コアフロー完成後に追加する |
| 通知 | DM・チームトークと合わせて後から実装する |
| 実績機能 | MVP のスコープ外 |
| いいね機能 | 申請機能で代替できる |
| 募集要項の充実 | 最初はシンプルな投稿フォームで十分 |
| コミュニティ規約 | 運用しながら調整が必要 |

## MVP 必須画面

| 画面 | 説明 |
|------|------|
| 案一覧 | プロダクト案の一覧（デフォルト新着順） |
| 投稿作成 / 編集 | 新規案の作成・既存案の編集 |
| 投稿詳細 | 案の詳細・ポジション一覧・参加申請ボタン |
| マイページ | 自分の投稿・参加プロジェクト・受けた申請の管理 |
